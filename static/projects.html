I like to make things.

Here are some of them.

Code on GitHub

  • gapi is a command-line client for Google APIs.
  • simply-put is a general-purpose REST+JSON backend API for the building of future toy apps.
  • gifexplode splits an animated GIF into its constituent frames.
  • perlerize generates Perler bead templates from PNG images.
  • csvstruct makes it easy to generate CSV data from Go structs
  • tros allows you to sort slices of structs using reflection in Go (get it?)
  • go-marvel is a Go client library for the Marvel API, which lets you do cool stuff
  • spam-me lets you give out an email address without fear of spam.
  • pointer-party lets your mouse pointer get out there and live a little.
  • go-gist makes it easier to import and use Go code from GitHub Gists.
  • imjasonh.com runs imjasonh.com, which you're looking at right now

Code on code.google.com

Generally speaking, this is older code that may or may not still work anymore.

  • chatwithstrangers lets two random people chat using XMPP.
  • mrjs is an experiment in running distributed code in strangers' browser, and is a pretty terrible idea as security is concerned.
  • chrome-4sq was a Chrome extension to notify you of your friends' Foursquare checkins, but hasn't been shown any love in a long long time :(
